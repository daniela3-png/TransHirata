# Sistema de Gestión Logística - Transportes Hirata

Software de escritorio desarrollado en Java para la gestión integral de flota, mantenimiento preventivo y control de conductores.

## Funcionalidades Principales
- **Control de Acceso:** Login con roles (ADMIN, MANTENCION, CONDUCTOR).
- **Gestión de Flota:** CRUD de vehículos y monitoreo de estado (Activo/Taller).
- **Mantenimiento Automatizado:** Bloqueo de vehículos cada 5.000 km.
- **Historial de Servicios:** Reportes detallados con trazabilidad de solicitantes.

## Tecnologías Utilizadas
- **Lenguaje:** Java 17+ (Maven)
- **Base de Datos:** MySQL 8.0
- **Interfaz:** Swing (JFrame)
